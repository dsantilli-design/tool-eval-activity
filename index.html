AI Tool Evaluation Framework

1. AI Behavior Type

  • Generative – Creates new content
  • Predictive – Forecasts outcomes
  • Assistive – Aids human tasks
  • Adaptive – Adjusts to learner needs

2. Inputs Used

  • Text
  • Image
  • Audio
  • Structured data
  • Student data

3. Output Type

  • Draft
  • Decision recommendation
  • Analysis
  • Public-facing content

4. Risks

  • Hallucination
  • Bias
  • Privacy
  • Overconfidence
  • Synthetic realism

5. Oversight Required

  • Light review
  • Careful fact-checking
  • Professional validation
  • Not appropriate
